# example_base

Base example template. Meant to be copied and modified to create new examples.

## Pre-requisites

- git
- cmake
- A C++ compiler toolchain

## Building

1. `mkdir build`
2. `cd build`
3. `cmake ..`
   This fetches Arrow from GitHub so it takes a bit and produces no output
4. `cmake --build .`
5. `./example`
